Module Thelia : TinyMCE 4 & Responsive Filemanager 9

Installer & Configurer

Ajouter le module dans Configuration » Activation des plugins de l'administration Thelia.

Attention: Si vous utilisez TinyMCE Plus, n'oubliez pas de le Désactiver. En revanche, il n'est pas nécessaire de le supprimer.

Réglages du TinyMCE 4
Fichier: /client/plugins/tinymce4/tinymce4_admin_title.php
Documentation: http://www.tinymce.com/wiki.php/Configuration

Réglages de Responsive Filemanager 9
Fichier: /client/plugins/tinymce4/filemanage/config/config.php
Documentation: http://www.responsivefilemanager.com

Réglages de Photo Editor by Aviary
Responsive Filemanager dispose de l'éditeur d'images Aviary. Pour l'utiliser, vous devez être en ligne (et non sur un http://localhost/) et vous devez vous enregistrer sur le site Aviary.com (il semblerait que cette dernière étape soit facultative).
L'offre gratuite est limitée Ã  une résolution 800x600 maximum - ce qui est plus que suffisant ;)
Une fois enregistré, vous devez saisir vos clés d'enregistrement dans le fichier config.php de Responsive Filemanager (section AVIARY config).

Packages utilisés

  • TinyMCE version 4.0.12
  • Responsive Filemanager version 9.3.2
